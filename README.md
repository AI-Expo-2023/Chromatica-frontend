# Chromatica (크로마티카) 란?

> 💡 그림을 못 그려도 OK! AI에게 맡겨봅시다 🤖

---
### 프로젝트 실행 방법
**준비하기**
아래 Shell 명령어를 순서대로 입력해 주세요.
> yarn 

아래 명령어로 dev 환경을 실행해 보세요!
> yarn dev
